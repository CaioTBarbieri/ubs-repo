## Descrição

Este repositório contém o sistema UBS desenvolvido para atender às necessidades de gestão e organização de Unidades Básicas de Saúde. O projeto integra múltiplas tecnologias e possui uma interface moderna e responsiva.

## Tecnologias Utilizadas

O projeto é composto por diferentes linguagens e ferramentas, entre elas:

- **Java** (48.1%): Lógica de negócio, backend (provavelmente usando frameworks como Spring Boot).
- **TypeScript** (24.9%): Desenvolvimento front-end, possivelmente utilizando Angular, React ou similar.
- **HTML** (21.5%): Estrutura das páginas web.
- **SCSS** (3.7%): Estilização avançada e modular do front-end.
- **CSS** (1.8%): Estilos complementares das páginas.

## Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/CaioTBarbieri/ubs-repo.git
   ```
2. **Acesse o diretório do projeto:**
   ```bash
   cd ubs-repo
   ```
3. **Siga as instruções de configuração para backend e frontend:**
   - Para Java/backend:
     - Certifique-se de ter o Java instalado.
     - Compile e suba o serviço com seu gerenciador de dependências (ex: Maven ou Gradle).
   - Para frontend (TypeScript):
     - Certifique-se de ter o Node.js instalado.
     - Instale as dependências:
       ```bash
       npm install
       ```
     - Rode o projeto:
       ```bash
       npm start
       ```

## Estrutura do Projeto

O repositório segue uma estrutura modular, separando responsabilidades de backend e frontend. A organização pode envolver pastas como `/src`, `/frontend`, `/backend`, mas recomenda-se consultar a árvore de diretórios ou a documentação técnica para detalhes.
